* Columns: 26
#### File Format
* CSV
#### Data Description
***The dataset contains structured vehicle listing information including:***
* Region
* Price
* Year
* Manufacturer
* Model
* Condition
* Cylinders
* Fuel
* Odometer
* Title status
* Transmission
* Paint color
* Drive
* Size
* Type
* State
* VIN
* Country
* Lat
* Long
* Posting_date
## Data Limitations & Considerations:
* Dataset last updated in  2021.
* May not reflect current used car market conditions.
* Contains missing values in multiple columns.
* Possible duplicate or inconsistent listings.
